Surgimento da internet

Voltar

A internet foi criada em 1969, nos Estados Unidos. Chamada de Arpanet, tinha como função interligar laboratórios de pesquisa.

O surgimento da internet se deu pela necessidade de se comunicar de uma forma rapida, e apartir desse conceito, outros foram sendo descobertos e aprofundados por exemplo: Como tornar essa comunicação segura, Como tornar a comunicação mais rápida, Ou como realizar troca de informaçoes e dados entre dispositivos em longas distâncias

Como funciona o HTML

Voltar

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Ele define o significado e a estrutura do conteúdo da web. O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web.

HTML consiste em uma série de elementos que você usa para anexar, envolver ou marcar diferentes partes do conteúdo para que apareça ou aja de uma certa maneira.

Oque são tags

Voltar

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Existe uma enorme diversidade nos tipos de elementos, podendo ser: Elementos básicos, Metadados do documento, Seção root, Separação de conteúdo, Conteúdo textual, Imagem e multimídia etc...
